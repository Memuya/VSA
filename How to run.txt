STARTING OFF
1. Download and install XAMPP.
2. Navigate to the "xampp" directory and find the folder "htdocs" and create a folder called "vsa".
3. Place all files inside the "source" folder on this repository into the "vsa" folder.
4. Start the XAMPP server (instructions below)
5. Go to "localhost/phpmyadmin/" in your browser.
6. Create a new database called VSA.
7. Import the sql file into the VSA database.

START XAMPP
1. Go to the "xampp" directory
2. Click on "xampp-control". A window will appear.
3. Press Start for "Apache" and "MySQL"
3. Go to "localhost/vsa/" in your browser to view the site
